openSUSE Taiwan 社群
openSUSE 正體中文社群創立於 2011 年，於 COSCUP 2011 結束後由幾位 openSUSE 攤位工作人員發起，社群目標是在台灣推廣 openSUSE，主要協助翻譯 Release notes 與 Wiki 條目，及不定期舉辦 Release Party 與 InstallFest... 等等。社群更在 2015 年舉辦了 openSUSE.Asia Summit 2015。openSUSE 有方便的 Yast 工具和 one click install 安裝套件，另外也提供了 SUSE Studio 讓喜愛客製化的朋友打造適合自己的 distribution，歡迎各位舊雨新知加入 openSUSE.zh-TW 一起推廣跟討論 openSUSE ！
