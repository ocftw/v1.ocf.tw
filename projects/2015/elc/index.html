OCF 國際交流分享會 - 資助學生向世界一流高手近身學習

原：OCF 國際交流計畫 - Embedded Linux Conference 分享會


● OCF 國際交流分享會 - 05/24 - 資助學生向世界一流高手近身學習 - 2015/05/24 14:00~17:00 @摩茲工寮/台北市中正區八德路一段 94 號 3F

● OCF 國際交流分享會 - 06/04 - 資助學生向世界一流高手近身學習 - 2015/06/04 19:00~21:00 @交通大學計算機中心一樓訓練教室 (近交大北門)


「哥教的不是知識是 guts！」 

    今年3月份由 Linux Foundation 所舉辦的 Embedded Linux Conference（ELC），黃敬群老師 (Jserv) 已經連續五年應邀參與演講，其演講的經驗與內容已備受肯定，這次特別地方在於 Jserv 也帶著成大學生廖健富 (Jeff Liaw) 一同前往發表開放原始碼成果和演說。在大學傳授課程的 Jserv ，一直希望能透過課堂上的教學，鼓勵同學能累積更多開放原始碼的創作，並且要能夠勇於站上舞台向眾人分享開發成果，達到自我能力的提升！


(ELC活動發表演說照片/ Jserv)

 

    05/24 活動特地邀請到 skymizer.com 的 ggm 和 Peter Chang 來分享參與國際大型活動的經驗，兩位從學生時代就已經積極參與各種活動，把握難能可貴的機會和世界一流的高手們相互切磋，而這些經驗開拓了彼此的國際視野。現在兩人與幾位好友一同創業，除了開發公司內部產品，依然不忘參與 Open Source 專案，他們深信，透過雙手的參與，可以讓軟體界變得更好。

 

OCF國際交流計劃：

廖健富 (Jeff Liaw)

 

國際交流計劃其目的為鼓勵台灣學生多多參與國際的交流活動，藉由資助旅費的方式，促使學生投稿國際的大型開源會議，直接和國際開源社群分享創作成果與經驗交流，並在回國後舉辦至少一場的分享會，讓有興趣的人也能來獲得新知和交流。

   Jeff Liaw 是第一位拿到 OCF 的國際交流計劃第一筆經費的受益者，經由此計劃獲得約 NT$30,000 旅費補助的學生，而他也依照與 OCF 國際交流計劃的約定，將會與 Jserv 一同向大家分享出國發表的題目：Optimize uClinux for ARM Cortex-M4 ，活動中也會分享參加ELC的見聞與拜訪舊金山開源愛好者的際遇，另外，他們也開放讓聽眾點播想聽的內容！如果你有想多聽點什麼議題，或你有什麼想問 jserv / Jeff Liaw 的地方，都可以在我們的表單內勾選或填寫。

05/24＠台北摩茲工寮 活動流程(14:00-17:00)：

* 介紹 OCF
* 技術分享 (30min) - Optimize uClinux for ARM Cortex-M4 (jserv and 廖健富)
* 茶點 / 交流時間 (30min)
* 見聞分享 (60min)
        * Embedded Linux Conference 見聞分享 - 30min (jserv and 廖健富)
        * Linux Symposium 經歷分享 - 30min (ggm and Peter Chang)

主辦單位：OCF開放文化基金會

    財團法人開放文化基金會致力於推廣各類開放文化活動，包括: 開放源碼、開放標準、開放硬體、開放資料、開放系統等相關領域之活動。

  • 提供開放源碼相關社群及活動專戶、收取款行政事務
  • 提供各社群贊助特定開發、議題研究專案的財務管道
  • 提供社群商標及法律保護
  • 提供國外非政府法人的對應窗口，對開放、開源有需要進一步聯繫者
  • 針對政府科技制度、法律相關議題提出研究報告及建議
  • 推動開放源碼於各領域的應用（如: 政府部門、非營利組織、業界…）
  • 基金會自己應用、驗證過的技術，可輸出至其他政府、非政府組織
