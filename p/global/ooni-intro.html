網路干擾，包含網站存取的阻擋、插入惡意程式碼等等，是現今各國政府用於控制資訊的手段之一，最知名的網路干擾計劃，莫過於中國的防火長城，封鎖中國境內不得存取維基百科、 Google, BBC 等網路上重要的資訊和消息來源，並偵測及封鎖各種翻牆軟體。除了防火長城之外，各國政府或多或少均有使用各種技術封鎖不同網站，理由可能包含著作權侵害、散佈假消息、散佈不利於政府的資訊等等。網路封鎖阻擋了資訊的自由流通，經常也會影響到人民的言論自由、知情權、祕密通訊自由等基本權利。瞭解網路封鎖的狀況即可瞭解一部分政府在網路上侵犯公民權利的狀況。

公民要如何瞭解網路封鎖的現狀和趨勢呢？要瞭解政府阻擋了哪一些網站，是否有一套自動化的檢查程式？政府封鎖與開放特定網站，也會因為國家、地區、電信商、時間有所不同，是否有方法彙整不同條件下檢查程式的結果，讓我們看出趨勢？

解答以上這些問題，即是「網路干擾開放測量計劃」（Open Observatory of Network Interference, OONI）成立的初衷。

什麼是 OONI ?

OONI 是全世界最大的公開測量、研究網路封鎖的計劃，也是 Tor Project （洋蔥路由器）旗下的自由軟體專案。OONI 計劃涵蓋一套跨手機/電腦平臺的網路封鎖偵測程式「OONI Probe」，以及後端的資料庫和分析工具，可以將蒐集到的偵測資料，彙整進他們旗下的全球網路審查情形資料庫，內容包含：

各國政府封鎖網站的情況
各國政府禁用即時通訊軟體的情況（例如：Telegram）
各國政府禁用反封鎖工具（翻牆工具）的情況（例如：Tor）
網路速度與效能
網路中可能存在的審查/監視系統（中間盒）


自 2012 年起，OONI 已經從全球 200 多個國家，收集了數百萬筆網路測量資料，所有資料都發佈在 OONI Explorer 和 OONI API 上。

開放文化基金會為 OONI 的正式夥伴，除了在辦公室設置 OONI 偵測器之外，也將共同改進 OONI 網路干擾的測試清單、協助分析 OONI 收集的網路干擾測量資料。

瞭解台灣的網路干擾現狀

個人貢獻者也可以協助 OONI ！最簡單的貢獻方法，就是下載安裝 OONI Probe 應用程式（支援主要電腦及手機平台），在此應用程式內可以執行各種網路干擾測試，並收集資料回傳給 OONI 資料庫。

!!注意!! 測試執行時 OONI Probe 會嘗試連線至敏感的網站，例如盜版、色情、反政府相關網站，可能有法律風險，請自行瞭解所在地區法令，評估風險之後再執行測試。

使用 OONI Probe 檢測網路干擾時，將會依照裝置的所在國家，選擇該國家的測試清單，再加上國際測試清單，從當中隨機選擇數個網站，進行連線測試，並向資料庫回傳測試結果。除了預先定義的測試清單外，使用者也可以自行指定想要檢測的網站。

執行 OONI 檢測時，將會收集的資料包含：

裝置所在的網路商識別碼 (Automonous System Number)
國家
裝置使用的網域名稱伺服器 (DNS Server) IP
DNS 查詢結果
TCP 連線結果
網頁回傳內容
其他：完整的資料項目可以到 OONI Explorer 的原始資料區查看

執行 OONI 檢測所收集的資料將有助於研究人員瞭解網站是否真的遭到干擾，以及干擾所使用的技術。

除了網頁連線的干擾之外，OONI Probe 也可以檢測針對常見的通訊軟體 (Facebook Messenger, Telegram, Whatsapp) 、翻牆軟體 (Tor, VPN) 的網路干擾。

群眾協力測試

疑似網路封鎖事件發生時，現有的測試清單內可能不包含新近被封鎖的網站，因而沒有新網站的網路干擾資料。為彌補此缺陷，OONI 提供了 OONI Run ，任何人都可以到 OONI Run 輸入想要檢測的網站位址，這將產生一個可供分享的檢測超連結，當有人點開這個超連結，他們裝置上的 OONI Probe 將自動啟動，開始檢測超連結中指定的網站。若裝置上沒有安裝 OONI Run，他們將被引導至安裝指引頁面。

舉例來說，假設某國家的人懷疑他們國內開始封鎖 https://example.com 網站，他們可以至 OONI Run ，並輸入 https://example.com ，這將產生一個檢測超連結 https://run.ooni.io/nettest?tn=web_connectivity&ta=%7B%22urls%22%3A%5B%22https%3A%2F%2Fexample.com%22%5D%7D&mv=1.2.0 。他們可廣為分享此檢測超連結，點開超連結的裝置將會啟動 OONI Probe ，並開始針對 https://example.com 進行網路干擾測試。

瞭解檢測結果

使用者可以到 OONI Explorer 的搜尋頁面瞭解 OONI 在各國所收集到的網路干擾現狀，可以針對國家、時間區間、網站、檢測類型、檢測結果進行搜尋。
